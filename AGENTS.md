# AGENTS.md

## Project Theory: Transcript-First Annotation

Transcript is the primary artifact. Annotations are secondary, anchored to canonical IDs.
Podcast players optimize playback. We optimize understanding.

Transcript-first conversation interface. Audio/video are optional synchronized lenses.
Marginalia culture over forum culture — passage-anchored, not global feed.
Readability and comprehension over ornamentation. Quiet motion, comfortable density.

## Contracts
1. Deep-links: `#t=<seconds>`, `#msg-<seconds>`
2. Linkability units: message anchors, heading anchors
3. Annotations must reference canonical anchors — no free-floating objects
4. Transcript cleanup: remove disfluency/filler only. No paraphrase, no added claims, no speaker drift

## Scope
Frontend iteration on homepage/episode reading experience.
No backend/auth/comments. No transcript reprocessing.
